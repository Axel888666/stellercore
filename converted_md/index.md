# Redoop Enterprise V9.1 重磅发布

## 大数据基础软件平台

Redoop Enterprise（代号"CRH"）是大数据基础软件，基于 Apache
Hadoop、Apache Hbase 为存储和计算核心，以Apache Ambari为管理平台，支持
Hive、Spark、Kafka、Flink、Doris、Hudi
等开源大数据生态软件，支持界面化安装、配置、监控各种大数据软件。
提高交付效率，降低软件故障，降低学习和维护成本，整体提升企业大数据运行效率。
